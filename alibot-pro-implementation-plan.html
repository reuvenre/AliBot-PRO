01

סקירה ומטרות

ALIBOT PRO הוא פלטפורמת SaaS לניהול מלא של שיווק שותפים AliExpress לשוק הישראלי. המערכת מחברת בין ייבוא מוצרים, יצירת תוכן שיווקי בעברית באמצעות AI, ושליחה אוטומטית לטלגרם — הכל בממשק ניהול אחד.

חסכון בזמן

מ-3 שעות ליום של כתיבה ידנית לפחות מ-15 דקות — OpenAI מייצר פוסטים שיווקיים מותאמים לכל פלטפורמה בעברית טבעית ואנרגטית.

אוטומציה מלאה

תור שליחה חכם + תזמון אוטומטי לפי שעות — הפוסטים נשלחים לטלגרם בלי להיות ליד המחשב. מרווח הפוסטים, שעות השליחה — הכל ניתן להגדרה.

ייבוא מוצרים

מחיפוש AliExpress ישיר, URL, או Discover Page — עם נתוני מחיר, תמונה ועמלה בזמן אמת.

יצירת תוכן AI

OpenAI GPT-4o-mini מייצר פוסטים בעברית, עם בחירת פלטפורמה, טון, וריאציות מרובות.

שליחה לטלגרם

Telegram Bot API — שליחה ישירה לערוץ/קבוצה עם תמונת מוצר, קישור אפיליאט, וטקסט מעוצב.

מצב נוכחי — מה כבר קיים

המערכת כבר בנויה חלקית ומכילה תשתית מוצקה. רוב ה-Backend קיים. הפיתוח הנדרש הוא בעיקר שכבת ה-Frontend ושיפורי UX.

קטלוג מוצרים (Catalog)
רשימת מוצרים, סינון לפי סטטוס, חיפוש, אישור/דחייה, סנכרון מחיר, קישור אפיליאט. ✅ מלא
✓ מוכן
Discover Page — גלה מוצרים
Hot Products, חיפוש מוצרים לפי מילות מפתח, פילטרים, grid/list view, הוספה לקטלוג. ✅ מלא
✓ מוכן
תור שליחה (Queue System)
הוספת מוצר לתור, OpenAI מייצר טקסט, שליחה לטלגרם לפי סדר. ✅ מלא Backend + UI בסיסי
✓ מוכן
תזמון אוטומטי (Scheduler)
הגדרת שעות שליחה, מרווח בין פוסטים, Cron job כל דקה. ✅ Backend + SchedulingForm UI
✓ מוכן
Campaigns (קמפיינים)
יצירת קמפיין לפי מילות מפתח, תזמון cron, יצירת פוסטים אוטומטית. ✅ Backend + UI
✓ מוכן
Post Generation — בחירת פלטפורמה + טון
Backend קיים, אבל Frontend לא חושף אפשרות לבחור פלטפורמה (Telegram/WhatsApp/Facebook) או טון. נדרש UI.

06

זרימת משתמש — Flow מלא

01
גילוי מוצר (Discover) המשתמש נכנס ל"גלה מוצרים" → מחפש לפי מילות מפתח או Hot Products → רואה מוצרים עם עמלה, מחיר ₪, דירוג. לוחץ "הוסף מוצר" → נשמר לקטלוג עם כל הנתונים המדויקים (ללא re-fetch שגוי).
02
ניהול קטלוג המשתמש רואה את המוצרים שלו → מאשר/דוחה → לוחץ "הוסף לתור" → OpenAI מייצר פוסט + מוסיף לתור השליחה. אפשרות לייבוא ישיר מ-URL AliExpress.
03
עריכת פוסט (Phase 3) PostGeneratorModal נפתח → בחירת פלטפורמה (טלגרם/ווטסאפ/פייסבוק) → בחירת טון (מקצועי/כיפי/דחוף) → OpenAI מייצר 3 גרסאות → המשתמש בוחר, עורך ישיר, ולוחץ "הוסף לתור" או "שלח עכשיו".
04
תור וסדר שליחה המשתמש רואה את התור עם drag-to-reorder → מגדיר שעות שליחה ומרווח → Cron Job שולח פוסט כל X דקות בין שעת ההתחלה לסיום.
05
מעקב ואנליטיקס Dashboard מציג: פוסטים שנשלחו / עמלה משוערת / ביצועי קמפיינים. היסטוריית פוסטים מלאה עם תמונה, פלטפורמה, זמן שליחה.

10

מדדי הצלחה (KPIs)

זמן לפוסט ראשון

פחות מ-30 שניות מרגע לחיצת "הוסף לתור" ועד פוסט מוכן בממשק.

איכות פוסטים

80%+ מהפוסטים לא ידרשו עריכה — נמדד ב-Beta עם 10 משתמשים ראשונים.

שיעור Regenerate

פחות מ-25% מהפוסטים ידרשו "נסח מחדש" — מדד לאיכות ה-Prompt.

משתמשי Beta

10 משווקי שותפים ישראלים פעילים בשבוע הראשון לאחר MVP.

פוסטים ביום

ממוצע של 20+ פוסטים שנשלחו ביום לכל משתמש פעיל.

NPS

NPS > 50 לאחר 2 שבועות שימוש — נמדד בפידבק ישיר.
