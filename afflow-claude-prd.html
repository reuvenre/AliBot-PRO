01

סקירה כללית ומטרות

AffFlow Pro הוא Claude Artifact אינטראקטיבי שמשכפל ומשפר את הפונקציונליות המרכזית של Afflow.co – פלטפורמת אוטומציה לשיווק שותפים – תוך שימוש ב-Claude Sonnet 4 כמנוע ה-AI. המוצר מאפשר למשווקי שותפים ישראלים לייבא מוצרים מ-AliExpress, לייצר פוסטים שיווקיים בעברית לפי פלטפורמה (טלגרם, וואטסאפ, פייסבוק), ולנהל את כל הזרימה בתוך ממשק אחד – ללא קוד.

🎯 מטרת המוצר

לצמצם את הזמן שמשווקי שותפים מבלים על כתיבת פוסטים ידנית – מ-3 שעות ליום לפחות מ-15 דקות – באמצעות AI שמייצר תוכן מותאם לכל פלטפורמה בעברית.

✨ מה מייחד אותנו

Claude מייצר עברית טבעית עם אמוג'י מדויקים, קריאות לפעולה, ופורמט ספציפי לכל ערוץ – לא תבניות גנריות. הפוסטים נשמעים כאילו נכתבו על ידי מומחה שיווק ישראלי.

📦 מיובא מ-

AliExpress, קישור ישיר, או הזנה ידנית של שם + תמונה + מחיר

📤 מפורסם ל-

טלגרם, וואטסאפ, פייסבוק, אינסטגרם (עתידי)

📊 עם מעקב

קישורי אפיליאט + מעקב קליקים + UTM אוטומטי לכל פוסט

פרסונה ומשתמש יעד

AliExpress Admitad טלגרם Windows

הצטרפה לתחום לפני 3 חודשים. מנהלת קבוצת וואטסאפ ופייסבוק. מתקשה לכתוב טקסטים שיווקיים שנשמעים מקצועיים. רוצה כלי שיסביר לה מה לעשות שלב-שלב.

03

סקופ – מה אנחנו בונים

A/B Testing אוטומטי

ניהול חשבון + Authentication

תזמון פוסטים עתידי

גרירת תמונות / העלאת קבצים

04

פיצ'רים ועדיפויות

🔗
ייבוא מוצר
הדבקת URL מ-AliExpress → Claude מחלץ שם, מחיר, תיאור ומוצא קישור אפיליאט מוצע. גם הזנה ידנית אפשרית (שם + תמונה + מחיר + קישור)
HIGH
✍️
יצירת פוסט בעברית
Claude מייצר פוסט מותאם לפלטפורמה שנבחרה: טלגרם (קצר + אמוג'י), וואטסאפ (אישי יותר), פייסבוק (ממוקד מכירה). אפשר לבחור טון: מקצועי / כיפי / דחוף / כלכלי
HIGH
🔄
Regenerate + וריאציות
כפתור "נסח מחדש" שמייצר גרסה שונה של אותו פוסט. אפשרות להציג 2-3 וריאציות ולבחור
HIGH
📋
העתקה חכמה
כפתור "העתק לטלגרם" / "העתק לוואטסאפ" – מעצב את הטקסט בפורמט הנכון ומעתיק ללוח
MED
📦
היסטוריית פוסטים
שמירת כל הפוסטים שנוצרו בסשן הנוכחי עם תמונת מוצר, תאריך, ופלטפורמה
MED
🤖
גילוי מוצרים (AI Discovery)
Claude מחפש בזמן אמת (Web Search tool) מוצרים טרנדים עם עמלה גבוהה לפי קטגוריה שהמשתמש בוחר
FUTURE
05

זרימת משתמש

01
ייבוא מוצר המשתמש מדביק URL מ-AliExpress או מזין שם + מחיר + קישור אפיליאט. Claude מחלץ את המידע ומציג תצוגה מקדימה של המוצר.
02
בחירת פלטפורמה וטון המשתמש בוחר יעד: טלגרם / וואטסאפ / פייסבוק. בוחר טון: מקצועי / כיפי / דחוף (מבצע). אפשרי לבחור מספר פלטפורמות בו-זמנית.
03
יצירת פוסט על ידי Claude Claude API נקרא עם Prompt מותאם. הפוסט מוצג בטקסט מבוסס RTL עם אמוג'י ו-CTA. זמן תגובה צפוי: 3-5 שניות.
04
עריכה ואישור המשתמש יכול לערוך את הפוסט ישירות, לבקש "נסח מחדש" (Regenerate), או לבקש שינוי טון ספציפי.
05
העתקה ופרסום לחיצה על "העתק" מכינה את הפוסט עם קישור האפיליאט ו-UTM. המשתמש מדביק בוואטסאפ / טלגרם / פייסבוק. הפוסט נשמר בהיסטוריה.
06

מסכים ו-UI

הממשק מבנה כ-SPA חד-עמודי חלוקה ל-3 פאנלים עיקריים ב-React / HTML+CSS:

🏠 Panel A – ייבוא

שדה URL + כפתור "יבא מוצר" + הזנה ידנית (accordion). תצוגת כרטיסי מוצר עם תמונה, שם, מחיר

⚙️ Panel B – הגדרות

Toggle פלטפורמה (טלגרם/וואטסאפ/פייסבוק), בחירת טון, שדה קישור אפיליאט

📝 Panel C – פוסט

תיבת טקסט RTL עם הפוסט שנוצר, כפתורי "נסח מחדש" + "העתק" + "שמור להיסטוריה"

ארכיטקטורת Claude API

מודל

claude-sonnet-4-20250514
שימוש ב-Streaming לחוויה מהירה יותר

Max Tokens

500 טוקן לפוסט בודד · 1,000 לגרסה עם 3 וריאציות

Use Case טוקן קלט טוקן פלט עלות משוערת
פוסט בודד (1 פלטפורמה) ~400 ~300 ~$0.003
3 וריאציות לפוסט ~450 ~900 ~$0.007
ייבוא + חילוץ מידע מ-URL ~600 ~200 ~$0.004
100 פוסטים ביום (משתמש פעיל) – – ~$0.70/יום
🧠 System Prompt Design – עקרונות

1. Persona: "אתה מומחה שיווק שותפים ישראלי עם 10 שנות ניסיון. אתה מכיר את השוק הישראלי, את שפת הרחוב, ואת מה שמוכר."

2. Platform Rules: לכל פלטפורמה קיים prompt section עם הוראות פורמט (אורך, אמוג'י, hashtags, שימוש ב-HTML bold/italic לטלגרם).

3. Few-Shot Examples: 2 דוגמאות לכל פלטפורמה ישולבו ב-System Prompt.

4. JSON Output Mode: בקשת 3 וריאציות תחזיר JSON מובנה עם { variant_1, variant_2, variant_3 }.
